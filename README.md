## Run Locally

Clone the project

```bash
  git clonehttps://blog-website-tawny-beta.vercel.app/auth

Go to the project directory

For Frontend:
```bash
  cd client
  npm i
  nom start

```
For Backend:


```bash
  cd server 
  npm i
  node server.js
```
envfile setup
```bash
PORT
MONGO_URI
JWT_SECRET



```
